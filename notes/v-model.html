Using v-bind and v-on

together

, we can create two-way bindings on form input elements: template:

input :value="text" @input="onInput"

methods: {

onInput(e) {

// a v-on handler receives the native DOM event

// as the argument.

this.text = e.target.value

}

}

To simplify

bindings, Vue provides a directive, v-model, which is essentially a syntax sugar for the above:

{{text}}

{{v_text}}
